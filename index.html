SohoSuite is a tool suite for Windows designed to collect all of the tools available for developing and modifying the Amazon Kindle Fire HD (7") 2013 into one User Interface to make things as easy as possible for endusers.

SohoSuite is the 4.0 replacement for KindleSuite
SohoSuite is in current development. It is not recommended that you use this program unless you are a tester for SecretDevWork.

SohoSuite includes the following features:

  • Backup & Restore Apps
  • Factory Restore
  • Root
  • Install Gapps
  • Install ROM's
